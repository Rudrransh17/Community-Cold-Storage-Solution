# Community-Based Cold Storage Solution for Smallholder Farmers

## Overview

This project proposes a practical community-based cold storage facility designed to reduce post-harvest losses, improve farmer incomes, and strengthen rural agricultural infrastructure. The model is based on a shared storage system managed by a Farmer Producer Organization (FPO), making cold storage accessible and affordable for smallholder farmers.

## Objectives

* Reduce post-harvest losses from 15% to 5%.
* Improve storage accessibility for small farmers.
* Enable better market timing and reduce distress selling.
* Promote sustainable and climate-resilient agricultural practices.
* Assess the financial and operational feasibility of a community cold storage facility.

## Key Assumptions

* Farmers Served: 100
* Storage Capacity: 20 MT
* Annual Throughput: 400 MT
* Ownership Model: FPO Managed
* Power Source: Grid + Solar PV + Backup Generator
* Total CAPEX: ₹50 Lakh
* Annual OPEX: ₹7.5 Lakh
* Payback Period: ~5.3 Years
